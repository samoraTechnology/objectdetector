# VisionAI - Neural Object Tracking

A high-performance, browser-only React application for real-time object detection and tracking.

## 🚀 Features
- **TFJS Acceleration**: Uses TensorFlow.js for client-side neural inference.
- **COCO-SSD Model**: Detects 80 different object categories with high confidence.
- **Real-time Stats**: Low-latency HUD with FPS and inference timing telemetry.
- **Entity Filtering**: Narrow down detection to specific object classes.
- **Data Capture**: High-resolution snapshots with bounding box overlays.
- **Modern UI**: Dark-themed, glassmorphic interface with fluid animations via Framer Motion.

## 🛠️ Technology Stack
- **React.js**: Frontend framework (Functional components + Hooks)
- **TensorFlow.js**: Machine Learning engine
- **COCO-SSD**: Pre-trained object detection model
- **Tailwind CSS**: Modern utility-first styling
- **Lucide React**: Vector icons
- **Framer Motion**: State-driven animations

## ⚙️ Installation
1. Clone or copy the project files.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## 🔒 Privacy
All processing is performed locally on your device. No camera data is sent to any server. Everything remains 100% private and offline-capable once loaded.
